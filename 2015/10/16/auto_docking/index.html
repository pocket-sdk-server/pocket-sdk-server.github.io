自动化打包系统接入指南 V1.0

一、概述

本文档用于指导WEB客户端快速接入PocketGames自动化打包系统服务器端

二、代码接入详细介绍

1.查看游戏列表接口

  • 请求URL地址

    http://ip地址:端口/pocketgames/web/game/list

  • 请求参数

    HTTP协议，支持GET方法

  • 返回参数

    以JOSN格式返回结果

名称 类型 示例值 描述
code int 返回状态 200=成功,其他等于失败
error_msg String 错误信息
data Array 对象数组
data[i].appId int 应用ID
data[i].appName String 应用名称
data[i].icon String 应用图标路径
data[i].screen int 横竖屏 0=横屏 1=竖屏
data[i].onLineChannel int 在线渠道总数
data[i].allChannel int 所有渠道总数
data[i].appStatus Object 应用状态
data[i].appStatus.android int android应用状态 0=未接入 1=接入完成
data[i].appStatus.ios int ios应用状态 0=未接入 1=接入完成
